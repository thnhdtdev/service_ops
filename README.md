# ServiceOps

ServiceOps is a web-based operations management system for laundry and shoe care businesses. The application helps store staff manage daily orders, customers, service pricing, payment status, and simple revenue reports.

## Overview

This project is designed for small local service stores that need a simple and fast way to manage daily operations. Instead of using manual notes or spreadsheets, staff can create orders, track order status, manage customer information, and review business performance in one place.

## Main Features

- Fast order creation
- Automatic price calculation based on service type and quantity
- Order status management
- Payment status tracking
- Customer management
- Service pricing management
- Daily revenue summary
- Responsive dashboard for desktop and mobile

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- React Hook Form
- Zod
- TanStack Query
- Supabase
- Vercel

## Project Goal

The goal of this project is to build a practical full-stack web application based on a real business use case. It focuses on clean UI, simple workflow, reusable components, form validation, data management, and business-oriented frontend development.
